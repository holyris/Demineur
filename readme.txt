Pour windows :
     
	Si vous avez déjà Java d'installé sur cet ordinateur, il vous suffit juste de télécharger "Demineur jar" et de lancer Demineur.jar. Cependant un JRE de version 53.0 est nécessaire. Il est facilement téléchargable ici http://www.oracle.com/technetwork/java/javase/overview/index.html
	
Pour linux :
     
	Télécharger "Demineur linux" et suivez les instructions écrites dans le readme.txt présent dans le dossier.
